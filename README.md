# Age Calculator

A web application to calculate your exact age in years, months, and days.

## Features

- Custom JavaScript datepicker (Air Datepicker) – no default HTML date input
- Age calculation using [Luxon](https://moment.github.io/luxon/)
- Responsive design with [TailwindCSS](https://tailwindcss.com/)
- Input validation (invalid dates are handled)
- Clean and modern UI

## How it works

1. Enter your birthdate using the datepicker.
2. Click **Calculate**.
3. The app will display your precise age in years, months, and days.
